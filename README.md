# CYBER-MAN · Antivirus vs Malware

Juego arcade tipo Pac-Man con tema de ciberseguridad, hecho para ADAMA.

Eres el **Antivirus** y debes limpiar todos los datos infectados del laberinto
mientras esquivas al malware:

- 👾 **Troyano**
- 🪱 **Gusano**
- 🔒 **Ransom**
- 🕵️ **Spyware**

Recoge los **Parches** 💊 para activar el modo invencible y eliminar virus.

## Cómo jugar
- PC: flechas o teclas WASD
- Celular: desliza el dedo sobre el tablero

## Jugar online
Una vez publicado con GitHub Pages, el link queda disponible para cualquiera.
